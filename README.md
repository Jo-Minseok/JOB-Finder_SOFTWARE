# Job-Finder 소프트웨어

제작일: 2023-12-08
잡코리아 웹사이트에 영감을 받아 프로그램으로 만들어봤습니다.<br>

<br>
소스코드내 드라이버와 연결되는 IP 부분은 지워놨습니다.<br>
